# CodvedaTech Employee Management System

This project is an Employee Management System designed to manage employee records efficiently.

- Add new employees
- Update employee details
- Delete employee records
- View all employees
- Search employees

- Object-Oriented Programming (OOP)
- CRUD Operations
- Data Structures

## 🛠️ Technologies Used
- Java
- MySQL Workbench 8.0
